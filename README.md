 # 🎴 Card Matching Game (Python - Tkinter)

A GUI-based Card Matching Game developed using Python and Tkinter.  
The game challenges players to find matching pairs of cards in a grid.

## 🚀 Features
- Interactive GUI using Tkinter
- Card flip and match logic
- Move counter and timer
- Restart game functionality

## 🛠️ Technologies Used
- Python
- Tkinter

## 🎮 How to Play
1. Click on any two cards to flip them.
2. If the cards match, they remain open.
3. If not, they flip back.
4. Match all pairs to win the game.
